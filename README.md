# GURISES - Proyecto Final

## Objetivo general

GURISES es un proyecto de análisis de datos aplicado a la educación infantil y adolescente cuyo objetivo principal es acompañar la observación educativa desde una mirada integral, respetuosa y alineada con la pedagogía Montessori.

El proyecto parte de una convicción clara:
los datos y la tecnología solo aportan valor a la educación cuando se utilizan con responsabilidad ética, sensibilidad pedagógica y respeto por los procesos naturales del desarrollo humano.

En lugar de centrarse exclusivamente en el rendimiento académico o en métricas de éxito escolar, GURISES pone el foco en la relación entre el entorno educativo, la autonomía, el bienestar y las condiciones de base que hacen posible el aprendizaje profundo.

## Qué hace el proyecto 


El proyecto integra distintas etapas y herramientas de análisis de datos aplicadas a un contexto educativo real:

Análisis de datasets educativos relacionados con hábitos de estudio, entorno, bienestar y variables psicosociales.

Construcción de índices pedagógicos interpretables, diseñados para sintetizar dimensiones clave del desarrollo:

Entorno educativo (ISEE)

Autonomía y autorregulación (IAA)

Bienestar integral (IBE)

Índice de observación educativa

Análisis exploratorio y visualización de datos para identificar patrones generales, no diagnósticos individuales.

Uso de clustering no supervisado (K-Means) para agrupar perfiles educativos orientativos, evitando enfoques predictivos deterministas.

Desarrollo de una aplicación interactiva en Streamlit, pensada para familias, docentes y equipos educativos, que traduce los resultados técnicos en mensajes claros, comprensibles y pedagógicamente responsables.

- **`app.py`**: Archivo principal para la ejecución del proyecto.
- **`powerbi_dataset.csv` y `powerbi_dataset.xlsx`**: Conjuntos de datos utilizados para análisis y visualización.
- ** y `Factors1.ipynb`**: Notebooks de Jupyter para análisis exploratorio de datos (EDA).
- **`StudentPerformanceFactors.csv`**: Datos sobre factores que afectan el rendimiento estudiantil.

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/GURISES-ProyectoFinal.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd GURISES-ProyectoFinal
   ```
3. Instala las dependencias necesarias (asegúrate de tener `pip` instalado):
   ```bash
   pip install -r requirements.txt
   ```

## Ejecución

1. Ejecuta el archivo principal para iniciar el análisis:
   ```bash
   python app.py
   ```
2. Explora los notebooks para un análisis detallado y visualización de datos:
   - `Factors1.ipynb`

## Aplicación Interactiva

La aplicación desarrollada con Streamlit permite:

Explorar la estructura del dataset y sus variables.

Visualizar índices pedagógicos construidos a partir de observaciones educativas.

Analizar perfiles educativos orientativos mediante clustering.


La experiencia está diseñada para ser intuitiva, explicable y éticamente responsable.

Para ejecutar la aplicación interactiva, utiliza el siguiente comando:
```bash
streamlit run app.py
```

## Metodología

El proyecto se apoya en los siguientes principios metodológicos:

Análisis exploratorio de datos (EDA) como base de comprensión.

Construcción de índices sintéticos interpretables.

Clustering no supervisado para identificar patrones generales.

Ausencia deliberada de modelos predictivos supervisados.


Desde el punto de vista pedagógico, el proyecto se alinea con los principios definidos por la Asociación Montessori Internacional (AMI):

Observación antes que intervención.

El entorno como agente educativo.

El bienestar como condición de base.

La autonomía como proceso, no como resultado.
